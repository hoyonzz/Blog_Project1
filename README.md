# Blog_Project
- Django를 이용하여 블로그를 제작하는 프로젝트

## 1. 목표와 기능

### 1.1 목표
- 개발 관련 공부한 내용을 게시하는 블로그
- 개발 외 일상 내용을 게시하는 블로그

### 1.2 기능
1) Django Admin을 이용한 게시글 읽기 및 메인페이지 구현하기
2) 블로그 CRUD 기능 구현하기
3) 로그인/회원가입 기능을 이용하여 블로그 구현하기
4) 블로그 기능 외 추가 기능 작성 및 배포
